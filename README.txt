
# Simon Says Game

A memory-based game where players repeat an increasingly complex sequence of flashes.

## How to Play

1. **Start the Game:** Click "START" to begin.
2. **Watch the Sequence:** Pay attention to the flashing buttons.
3. **Repeat the Sequence:** Click the buttons in the same order as they flashed.
4. **Advance Levels:** Correctly match the sequence to progress to the next level.
5. **Game Over:** The game ends when a wrong button is pressed. Press "START" to try again.

## Files

- **index.html:** The HTML structure for the game interface.
- **app.js:** The JavaScript logic controlling the game.
- **styles.css:** The CSS file for styling (if applicable).

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/alamrehan1234/simon-says-game.git
   ```
2. Open `index.html` in your browser to play.
